Confidence Interval for a Population Ratio

Usage

prob.ci(n, x, alp = 0.05, dig = 4)

Arguments

n

Sample size

x

Number of successes in a sample

alp

Level of significance, Default: 0.05

dig

Number of digits below the decimal point, Default: 4

Value

None.

Examples

prob.ci(n=200, x=15)
