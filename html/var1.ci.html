Confidence Interval for a Variance

Description

Confidence Interval for a Population Variance

Usage

var1.ci(x, alp = 0.05, dig = 4)

Arguments

x

Data vector

alp

Level of significance, Default: 0.05

dig

Number of digits below the decimal point, Default: 4

Value

None.

Examples

x = c(20.0, 21.5, 20.9, 19.8, 22.5, 20.3, 23.6, 18.0, 23.3, 17.8)
var1.ci(x, dig=3)
var1.ci(rnorm(36))
