Conditional Probability

Description

Calculating the Conditional Probability

Usage

cprt(a, b)

Arguments

a

Event to be conditioned.

b

Event to be conditioning.

Value

None.

Examples

S = rolldie2(2)
A = subset(S, (X1+X2) >=4)
B = subset(S, (X1+X2) >=8)
cprt(B, A)
