corr.plot6 {Rstat}R Documentation

Six Scatter Plots

Description

Create 6 types of scatter plots.

Usage

corr.plot6(m1 = 60, s1 = 10, m2 = 60, s2 = 10, r = 0.7, r2 = 0.8,
  n = 50)

Arguments

m1

Mean 1. Default: 60

s1

Standard deviation 1. Default: 10

m2

Mean 2. Default: 60

s2

Standard deviation 2. Default: 10

r

Correlation Coefficient 1. Default: 0.7

r2

Correlation Coefficient 2. Default: 0.8

n

Number of samples. Default: 50

Value

None

Examples

corr.plot6()
corr.plot6(r=0.6, r2=0.9, n=100)
