Confidence Interval for a Mean (known variance)

Description

Confidence Interval for a Population Mean (known variance)

Usage

pmean.ci(xb, sig, n, alp = 0.05, dig = 4)

Arguments

xb

Sample mean, or sample data

sig

Population standard deviation

n

Sample size (unnecessary when sample data are given)

alp

Level of significance, Default: 0.05

dig

Number of digits below the decimal point, Default: 4

Value

None.

Examples

pmean.ci(xb=199.5, sig=5, n=50)
pmean.ci(xb=199.5, sig=5, n=50, dig=3)
pmean.ci(rnorm(100, 20, 3), sig=3)
