pprt {Rstat}R Documentation

Probability of an Event

Description

Calculating the Probability of an Event

Usage

pprt(x, n, prt = TRUE)

Arguments

x

Event in data frame.

n

Size of the sample space.

prt

Print output? Default: TRUE

Value

The probability of an event.

Examples

S = rolldie2(2)
B = subset(S, (X1+X2) >=8)
pprt(B, nrow(S))
