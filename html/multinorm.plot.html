multinorm.plot {Rstat}R Documentation

Multinomial PDF Plot

Description

Graphic Display of Multinomial Probability Distribution Function

Usage

multinorm.plot(ps, size)

Arguments

ps

Probability (or proportion) matrix with column for each group

size

Sample size

Value

Matrix of multinomial PDF

Examples

library(scatterplot3d)
ps = matrix(c(1,1,8, 1,5,4, 4,4,2, 1,1,1), nrow=4, ncol=3, byrow=T)
multinorm.plot(ps, 5)
